## SETUP INSTRUCTIONS FOR LINUX

### 1. Create Virtual Environment
python3 -m venv gan-env

### 2. Activate Virtual Environment
source gan-env/bin/activate

### 3. Install Dependencies
pip install -r requirements.txt

Alternative (manual installation):
pip install pillow torch torchvision matplotlib numpy

### 4. Install System tkinter (required for GUI)
sudo apt-get update
sudo apt-get install -y python3-tk

## RUNNING THE PROGRAM

### Activate Environment and Run GUI
source gan-env/bin/activate
python3 TKinterWindow.py

### Run Alternative Programs
# For test menu:
python3 TestMenu.py

# For clinical data generation:
python3 ClinicalSupplementGAN.py

# For training generic GAN:
python3 genericGAN.py

## VS CODE SETUP (for IDE integration)

### Configure Python Interpreter in VS Code
1. Press Ctrl+Shift+P (or Cmd+Shift+P on Mac)
2. Type "Python: Select Interpreter"
3. Choose the interpreter from gan-env: /home/jmartinez/school-projects/dhin-team-1/gan-env/bin/python

This will resolve import warnings and provide proper IDE support.
